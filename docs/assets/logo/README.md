# Desktop Nano Cortex Logo

Selected direction: `S05 Rotary Surface`.

This mark is based on the app's actual control-surface language: dark hardware panel, tone/rotary
controls, cyan live-state accents, amber secondary accent, green active state, and the small LED row
used throughout the frontend.

Current assets:

- `app-icon.svg` - source icon for desktop app bundle generation.
- `readme-lockup.svg` - horizontal README/project banner.
- `agenticflowx-icon.svg` - AgenticFlowX icon mark (static, no embedded CSS animation), copied from `afx-project/afx-workflow/assets/agenticflow_icon_minimal.svg`. Used in the README's "Built with AgenticFlowX" section; owned upstream by the AgenticFlowX project, not this app's own brand. Deliberately the static variant, not the animated `afx-mark.svg` lockup: animated SVGs can render mid-transition when GitHub snapshots them as a static image.

Archived exploration work lives in `../archive/logo-exploration/`.
