# Support

Desktop Nano Cortex is an unofficial, solo-maintainer project. Support is
best-effort — there are no response-time guarantees.

## Where to get help

- **Bugs**: open a
  [GitHub Issue](https://github.com/rixrix/deskop-nano-cortex/issues) using the
  bug report template.
- **Questions and ideas**: use
  [GitHub Discussions](https://github.com/rixrix/deskop-nano-cortex/discussions)
  if enabled on the repo; otherwise open an issue.
- **Security problems**: see [SECURITY.md](SECURITY.md) — never a public issue.

## Useful references

- [README](README.md) — setup, documented MIDI table, caveats
- [Hardware smoke runbook](docs/runbooks/hardware-smoke.md) — manual device
  verification steps
- [Windows build runbook](docs/runbooks/windows-build.md) — building and
  testing on Windows
- [PRIVACY.md](PRIVACY.md) — what the app does (and does not) collect

## Expectations

- USB MIDI behavior follows the documented Nano Cortex MIDI list; BLE-derived
  state is experimental and provisional.
- Not affiliated with or endorsed by Neural DSP.
